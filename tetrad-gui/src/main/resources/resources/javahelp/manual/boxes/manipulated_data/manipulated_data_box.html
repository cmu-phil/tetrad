Inside the Manipulated Data Box

An Instantiated Model box in the main workspace looks like this:



The manipulated data box is used to alter a data file. Inside the box you may:

  • Interpolate a Missing Value value (an asterisk) whereever values are missing in the data file.
  • Insert the modal value of a discrete variable wherever values of that variable are missing in the data file. (Better interpolations are coming).
  • Produce a data set with multiplications of cases.. Each case is multiplied by the number in the leftmost column of the Data file input to the Manipulated Data Box.

Once it has been run, the Manipulated Data box works just like any other Data Box

Note one programming oddity: if you wish to discretize a continuous variable, you must do so inside the Data Box that holds the data file,not inside the Manipulated Data box.
