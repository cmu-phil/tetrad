Continuous and Discrete Variables

Variables in Tetrad specify the range of data points that can be stored in a column. Two types of variables are used: continuous and discrete. Continuous variables accept real numbers as values; discrete variables accept string values from a specified list of values.

Discrete variables may look like continuous continuous variables if they contain string values that look like strings. In the data editor, it is easy to tell the two apart, since discrete variables are in columns with names ending in "-T."
