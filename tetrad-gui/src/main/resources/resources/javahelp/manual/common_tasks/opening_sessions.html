Opening Sessions

Sessions that have been previously saved out as ".tet" files can be loaded back using the "Open Session..." menu item in the main File menu.

You will be asked to locate the file on your hard drive. Once you do, click "Open."

Note that sessions saved using versions published since March 2005 are stamped with the version number of Tetrad they were saved out with and the date they were saved. (See Tetrad Versioning.) If you need a specific version of the program for some reason, all versions since March, 2005 can be launched using Java Web Start from the Tetrad download site, and versions previous to March, 2005 are stored in the archive directory inside the Tetrad donwload directory on that site and can be launched as executable jars.
