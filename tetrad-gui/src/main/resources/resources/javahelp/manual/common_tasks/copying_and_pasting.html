Saving, Cutting, Pasting

Nodes and subgraphs of nodes, either in the main session workspace or in any graph display, can be copied, cut, and pasted, either into the same workspace of graph or into some other workspace or graph.

To copy a section of a flowchart graph, first select the nodes you copy. The edges connecting those nodes will be copied as well.

Now select "Copy" from the Edit menu, or type Control-C. If you now select "Paste" from the Edit menu, or type Control-V, a copy of the selected nodes and edges will be pasted into the workspace slightly down and to the right of the original.

These selected nodes can now be dragged to the desired location.

If you select "Cut" (or type Control-X) instead of "Copy" (Control-C), the original nodes will be deleted, but you will still have the option of pasting copies of them into the workspace.
