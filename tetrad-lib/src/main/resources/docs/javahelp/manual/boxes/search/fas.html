Search Algorithms: FAS

The FAS ("Fast Adjacency Search") algorithm is the adjacency search of the PC algorithm (see). It produces an undirected skeleton, with parents divorced. It is used in a number of algorithms included in this package.

 

References:

Spirtes, Glymour, and Scheines (2000). Causation, Prediction, and Search.
